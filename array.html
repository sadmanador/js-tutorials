pop()

pop() extracts the last element. the original arr changes with one less element

shift()

shifting is opposite of the popping, removes element from start. removing also happen from original arr. like pop, arr.shift() to see what is removed.

unshift()

adds a new element at the start of an array. quite similar as push() but adds at the beginning.

concat

splice()

  • adds a new item to an array
  • first parameter is position for insertion items.
  • second parameter defines how many old items will be removed. therefore, splice can also be use for removing items.
  • rest of the value parameter adds new items.

slice()

  • slice cut item to make a new array.
  • query is num of the item from start, not index.
  • query slice(2) means 2 items from start.
  • second parameter define what should be retained.
  • original array retain its value.
  • new array consist value except the cut values
