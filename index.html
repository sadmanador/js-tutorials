string slice()

name.slice();
slice cuts string from beginning and end, takes only middle value. slice takes two values. first one represent the starting point, second one defines the beginning of non-included letters first letter. when values are negative string should be counted from end. single value represent the start of string, rest is taken.

substring()

substring are similar to slice, however less than 0 values are considered to be 0.

substr()

similar to slice, however second value represents number of character should taken.

replace()

  • Should be used to make only new variable. returns a new string.
  • does not change the old string
  • only replaces the first match.
  • replace() is case sensitive.

to replace all the match global /text/g is used

String search

indexOf()

indexOf() match a letter or word, then return index position. indexOf() and lastIndexOf() takes a second parameter as index of search starting position.

lastIndexOf()

returns the last occurring of a specific letter or word.

search()

search() is similar to indexOf() but a bit different from each other. search() doesn't take second position, where indexOf() cannot take powerful search value

match()

match() search a string and return results as array. more than one match will be returned. when global property isn't added returns single match. case insensitive search is done by /str/gi.

includes()

includes() returns a true when finds the match and false when doesn't. also takes a second parameter as index for searching starting point.

startsWith()

return true when string starts with match and false when doesn't start with match value. second parameter for start point

endsWith()

return true when ends with match value, also takes second parameter for start point.
