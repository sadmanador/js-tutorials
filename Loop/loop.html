Loop

  • for - loops through a block of code.
  • for/in - loops through a properties of an object.
  • for/of - loops through value of iterable object.
  • while - loops through a block code when a specified condition is true.
  • do/while - quite similar to while loop.

for Loop

loops run the code over and over until certain condition is met.

do/while

do/while is almost like while, instead of code block written in the {}, code block are written at the beginning in "do".

for Loop

    let txt = "";

    for (let i = 0; i < 5; i++){
        txt += i;
    }

for/in loop

    let arr = [1,2,3,4];
    let txt = "";

    for (let x in arr){
        txt += arr[x];
    }

for/of loop

    let arr = [1,2,3,4];
    let txt = "";

    for (let x of arr){
        txt += x;
    }

while loop

    let txt = "";
    let x = 0;

    while ( x < 5) {
        txt += x;
        x++
    }

do/while loop

    let txt = "";
    let x = 0;

    do {
        txt += x;
        x++
    }
    while (x < 5);
